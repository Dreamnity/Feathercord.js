## Get started

<CardGrid stagger>
	<Card title="Installation" icon="down-caret">
	    Installation is as easy as
        <Tabs>
			<TabItem label="npm">

			```sh
			npm install styrofoam.js
			```

			</TabItem>
			<TabItem label="pnpm">

			```sh
			pnpm install styrofoam.js
			```

			</TabItem>
			<TabItem label="Yarn">

			```sh
			yarn install styrofoam.js
			```

			</TabItem>
			<TabItem label="Bun">

			```sh
			bun install styrofoam.js
			```

			</TabItem>
		</Tabs>
		and can go as fast as 2 seconds
	</Card>
	<Card title="Start coding" icon="laptop">
	    Creating a new bot is as simple as
		```js
		const styro = require("./index");
        const bot = new styro({
	        login: {
		        token: 'TOKEN',
	        },
        });
        bot.on("messageCreate", async msg =>
	        if(msg.content.startsWith('!ping'))
			    return bot.channels[msg.channel.id].messages.create('Pong!')
        );
        bot.on("ready",async () => {
	        bot.users["@me"]().then(me=>
			    console.log("Logged in as "+me.username+"!")
			)
        });
        bot.on("close", data => console.log("Connection closed: ", data));
        bot.on('error',console.error);
		```
	</Card>
	<Card title="Similar syntax to Discord API itself" icon="setting">
		Because styrofoam.js is built to be the middleman between javascript and Discord API, It's syntax is directly same to Discord API
	</Card>
